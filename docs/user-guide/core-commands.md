# Основные команды

Это базовые команды, доступные в DeBot по умолчанию.

### `.help`

Показывает это справочное сообщение. Список команд генерируется динамически и включает в себя как основные команды, так и команды из всех загруженных для вашего аккаунта модулей.

### `.about`

Выводит краткую информацию о боте и его версии.

### `.setlang <код_языка>`

Устанавливает язык для текущего аккаунта.

-   **`<код_языка>`**: Двухбуквенный код языка (например, `ru`, `en`).

Все системные сообщения и ответы от модулей (если они поддерживают локализацию) будут отображаться на выбранном языке.
